# Daily Digests

Daily digests live in this directory, named `YYYY-MM-DD.md`, and are generated by the Director agent at 7am America/New_York. Each digest summarizes what the Quality, Discovery, and Director agents did in the previous 24 hours — PRs merged, PRs awaiting review, new Tier 3 planning items, and anomalies worth flagging. Jon reads this first thing in the morning to decide where to spend his review time. The index of all digests is maintained at `docs/agents/director/digests-index.md`.
